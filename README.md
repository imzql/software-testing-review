## 安装步骤
无需安装，只需克隆仓库即可：
```bash
git clone https://github.com/zqlit/software-testing-review.git
cd software-testing-review
```

## 使用说明
在本仓库中，您可以找到软件测试课程的复习 PPT 和相关资料。请按照以下步骤查看文件：
1. 克隆仓库到本地。
2. 打开您需要的 PPT 文件进行复习。
